webscreenshot
==================

## Installation

```
$ git@github.com:jnreynoso/webscreenshot.git
```
### Pre-requisites

    # Python 2.7
    # Invoke
    
### Install Invoke

    # pip install invoke
   
## Usage

`Invoke`
```
$ invoke install
```
`or`
```
$ invoke runserver

```
`or`
```
$ invoke install runserver
```
