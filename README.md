webscreenshot
==================

Installation
------------

```
$ git clone git@github.com:jnreynoso/webscreenshot.git
```
Pre-requisites
--------------

    # Python 2.7
    # Invoke
    
### Install Invoke

    # pip install invoke
   
Usage
-----

### Install webkit2png` ###
```
$ invoke install
```
### Start Web Server port 8080 ###
```
$ invoke runserver

```
### All (Install and Start Web Server) ###
```
$ invoke install runserver
```
